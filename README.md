## Морской бой  
  
**Стек:** React v18, MobX v6, TS v4, grommet v2  

**Функционал:**  
* возможность переключаться между игровым режимом и режимом формирования поля
* возможность удалять и добавлять корабли по клику  
* автоматическое заполнение поля по кнопке  
* отображение инфы по количеству кораблей на своем поле и поле противника  
* игровой режим: игра с компьютером  

**Потестировать онлайн:** <https://sea-battle-three.vercel.app/>  
  
**Как запустить локально:** 
```
npm install
npm run start
```
