## Морской бой  
  
**Стек:** React v18, MobX v6, TS v4, grommet v2  

**Функционал**  
Доступны два режима: режим формирования поля и игровой.  
В режиме формирования поля реализовано:
* автоматическое заполнение поля кораблями;
* добавление и удаление кораблей вручную по клику;
* валидация по количеству кораблей на поле: если оно не соответствует заданному, игру начать не получится.  
В игровом режиме реализовано:
* однопользовательская игра против бота;
* обновляемая информация по количеству убитых кораблей на своем поле и поле противника;
* подведение итогов игры: победитель, количество ходов, корабли, которые остались на поле победителя.

**Потестировать онлайн:** <https://sea-battle-three.vercel.app/>  
*Интерфейс адаптирован под десктопные экраны. На мобильном устройстве может отображаться некорректно* 
  
**Запустить локально:** 
```
npm install
npm run start
```
